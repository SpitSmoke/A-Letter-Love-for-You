# 💖 Projeto Romântico para Minha Namorada 💖

## Sobre o Projeto

Este é um projeto especial que criei para a pessoa mais incrível do mundo! 🌟 É um aplicativo em Java que inclui:

1. **Jogo de Perguntas** 🥰 - Um quiz para ver o quanto ela me conhece!
2. **Diário de Memórias** 📖 - Um espaço para registrar nossas memórias especiais juntos.

## Objetivo

O objetivo deste projeto é celebrar nosso amor e compartilhar momentos inesquecíveis. Espero que ela goste tanto quanto eu gostei de criar! ❤️

## Como Executar

Para rodar o projeto, siga os passos abaixo:

1. **Clone o repositório**:
   ```bash
   git clone https://github.com/seuusuario/seurepositorio.git

2. **Navegue até o diretório**:
   ```bash
   cd seurepositorio

3. **Compile e execute o projeto**
    ```bash
    javac Main.java java Main

  **Funcionalidades**
  Jogo de Perguntas: Teste seus conhecimentos sobre nós dois!
  Diário de Memórias: Adicione e visualize memórias importantes que criamos juntos.
